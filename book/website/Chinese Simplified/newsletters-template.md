# 撰写时事通讯模板

下面概述的模板是我们自2019年9月以来一直用来为 _Ting Way_ newsletter收集新闻项目的指南。 这个模板已经经过了几个月的演变，绝不说它是“完美的”格式。 当重用此模板时，您应该尝试什么对您的社区有效。 你在前面的章节中提出的建议是非常受欢迎的。
