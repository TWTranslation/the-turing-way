# 認識された長所と短所。

リモートコラボレーション、作業、およびイベント参加に適用される、すべての人にとって同じ設定が常に動作することは言うまでもありません。 多くの人にとって、例えば、彼らが小さなアパートに住んでいるとき、それは必ずしも物理的なオフィスで働くほど効果的ではないかもしれません。 大家族と自分の居場所を共有し、遠隔作業の設定が不十分であるか、同僚とのつながりが薄いと感じます。 しかし、他の人にとっては、リモートワークがより楽しく、生産的である可能性があります。
> 「12年間、1日4時間通勤していた人として。 私は4つの単語で私の経験を合計することができます - *遠隔作業は不平等を減らします*", 私たちの貢献者の一人、パブロ・ロドリエ・サオーンチェス(GitHub: PabRod)は言っています。

この章では、 "pro"と"cons"セクションの下で他の人とリモートでコラボレーションした経験について、 *軽い意見* を紹介しました。

## Pros

- リモートコラボレーションにより、快適に旅行できなかったり、オフィスの近くに住むことができなかった人々の生活の質が劇的に向上します。
- CO2排出量を削減することで、環境への良好な影響を与えています。
- 現場の費用に対処しなくても全体的な組織コストを削減することができるので、低資源環境でも同様に確立されたフォーマットである。
- 分散チーム形式は、異なる時間帯、異なる職場の要件、または柔軟な労働時間の手配で最善を尽くす人々から人々を魅了します。
- これは、一般的に社会的相互作用と社会的ダイナミクスを確立する革新的な方法につながる可能性があります。
- それは、私たちが世界のさまざまな場所からより多くの同僚について学び、私たちの職場環境で作るために私たちの文化の違いと宿泊施設についてもっと私たちを教育するのに役立ちます。
- 意思決定は、その場所に関係なく、すべてのステークホルダーが関与することにより、より効率的になります, 会議は、相互に便利な時期に行われます.

## Coins

- 友情の開発は、私たち自身でこれを開発するのに慣れているので、難しいかもしれません。
- 働く生活と個人の生活の境界はより曖昧になります。
- オンライン会議を実行している場合 割り当てられた時間では妥結できないかもしれないしチームの決定の誤った解釈につながることがあるかもしれない
- 割り当てられた時間帯でほとんどの相互作用が発生する場合、競合解決が困難な場合があります。
- 場合によっては、そのような相互作用は、ボディランゲージやトーン(オンラインチャット)が不足しており、誤解や不必要な紛争につながる可能性があります。
- 偶然の相互作用や水クーラー効果やそれらが生み出す革新的なアイデアを失う可能性があります。
- たとえば、特定のタスクの複製に関する誤った仮定につながるハードウェアへのアクセスが制限されているなど、人々は異なる作業条件を持つことになります。
- 何人かの人々が集中し、訓練された仕事の練習を有することを困難にすることができる仕事のための十分な取り決めなしで忙しい共同作業スペースまたは家から働けば混乱するかもしれない。
