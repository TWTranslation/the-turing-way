# 貢献記録

*コントリビューター `。 d <code>` file and the contributors table in the `README file` with _The Teuring Way_.*

_mañana: Vía Turística_ usando [todos los colaboradores del bot](https://allcontributors.org) actualizamos la tabla de colaboradores con el nombre de cada persona donde las teclas emoji indican las diferentes tareas que han realizado (ver el archivo [README](https://github.com/alan-turing-institute/the-turing-way/blob/main/README.md#contributors)). 私たちは、異なる貢献が人々に異なることを意味し、彼らの個人的な利益に向かって異なる翻訳があることを理解しています。 スキル開発、価値交換、キャリアの高度化。 したがって、 [貢献者.md](https://github.com/alan-turing-institute/the-turing-way/contributors.md) ファイルを、 _The Turing Way_ コミュニティメンバーからの個人的なハイライトをキャプチャする専用の場所として提供します。

個々の貢献者は、「 _The Teuring Way_ コントリビューターからの個人的なハイライト」セクションの下で彼らの詳細を提供することを歓迎します。 組織のサポートとコラボレーションは、「コラボレーション組織」セクションに一覧表示されます。 各組織名と詳細が別々にリストされ、その組織からの各貢献者の貢献の詳細が表示されます。

あなたの仕事について公平に認められる方法については、 [コミュニティハンドブック](https://the-turing-way.netlify.app/community-handbook/acknowledgement) を参照してください。

## _La Vía Turínea_ Cancelaria

# 協働組織とプロジェクト

*チューリング・ウェイ・コミュニティは、雇用主、プロジェクト、または組織からの参加によってサポートされているメンバーから、親切な貢献を受け取ります。 このような貢献は、1つのプロジェクトまたは組織からの1つまたは複数のメンバーが協力して、チューリングウェイでリソースを構築し、維持する場合に適用されます。 これらの貢献には、The Turing Way リソースを基にしたプロジェクトや、さまざまな能力で The Turing Way チームメンバーとコラボレーションするプロジェクトも含まれます。 私たちは、これらの貢献者のそれぞれを個別に認め、「コラボレーション組織とプロジェクト」にそのプロフィールを掲載します。*

## [デルフト工科大学 - 応用科学](https://www.tudelft.nl/en/faculty-of-applied-sciences)

応用科学部はデルフト工科大学の中で最大の規模を誇り、社会が直面するいくつかの問題に対する革新的な解決策を見つけることに重点を置いています。 社会全体で広く利用できる技術開発を支えるために必要な基礎知識の開発。 この知識をより広い社会と相互に共有できるようにする。 教員は、データとコードの共有を重視し、2020年以降の [データ管理方針](https://www.tudelft.nl/en/library/research-data-management/r/policies/tu-delft-faculty-policies/) を定めています。 この取り組みで 応用科学部からの貢献は、主に _チューリングウェイ_の再現可能な研究章に焦点を当てています。

## [Tú quieres el Centro de eScience](https://www.esciencecenter.nl/)

オランダeScienceセンターは、科学コミュニティのためのドメイン全体のソフトウェアと方法の開発と適用のためのオランダの国立ハブです。 彼らの主な目標は、さまざまなコンピューティング経験を持つ科学者が利用可能な電子インフラストラクチャの可能性を十分に活用し、そうでなければ到達不可能な科学的ブレークスルーを達成できるようにすることです。 オランダのeサイエンスセンターは、主にオランダの国立研究評議会(NWO)と国家e-インフラ機関(SURF)によって資金を供給されています。

El Centro Holandés eScience mantiene [su propia guía](https://guide.esciencecenter.nl/) para el desarrollo de software reproducible. eScienceセンターガイドの焦点は、 _The Turing Way_ と大きな重複を伴っているため、重複を避けることは理にかなっています。 eScienceセンターは、eScienceガイドに関連する分野で _チューリングウェイ_ に貢献しています。 eScienceガイドは、情報が他の情報が複製される場合に _チューリング方法_ を指す。

## [FAIR クックブック](https://fairplus.github.io/the-fair-cookbook/content/home.html)

FAIR クックブックは、研究者やデータマネージャーの専門家がデータを検索可能、アクセシブル、相互運用可能、再利用可能(FAIR)にするのに役立つオンラインリソースです。 FAIRPlus Cookbook は _The Turing Way_ プロジェクトとコミュニティ モデルに基づいています。 そして、FAIR要素、オーディエンスの種類、読み取り時間、難易度に応じた「レシピ」としてチャプターを提供します。
